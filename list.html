Full Stack Software Glossary

Software
The intangible collection of executable code, data structures, and supporting documentation that directs computer hardware to perform specific tasks
Hardware
The physical, tangible components of a computing system (like the CPU, RAM, and storage) that execute your code.
Computing
The execution of algorithmic processes, data processing, and mathematical calculations required to make a software application function
Programming
The technical process of designing, writing, testing, and maintaining code to instruct a computer on how to perform a specific task
